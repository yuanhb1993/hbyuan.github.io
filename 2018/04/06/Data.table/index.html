数据处理之Data.table包

发表于 2018-04-06 | 更新于: 2018-04-06 | 分类于 R语言 ， 数据处理 | 阅读次数: |
字数统计: 651 words

安装

install.packages(“data.table”)
library(data.table)

Data.table 与data.frame可互相转换

dt <- data.table(a=c(‘A’,’B’,’C’,’A’,’A’,’B’),b=rnorm(6))
dt2<-data.frame(dt)

df<-data.frame(a=c(‘A’,’B’,’C’,’A’,’A’,’B’),b=rnorm(6))
df2<-data.table(df)

语法

DT[I, j, by]
I= 筛选
J= 运算函数
By=分组

keyby可根据设定分组排序，同时设定key

查询

  • 取第二行的数据

    dt[2,]

  • 不加,也可以

    dt[2]

  • 取前两行

    dt[1:2]

  • 取a列的值

    dt[, a]
    或
    dt$a

  • 取a列值，且返回data.table

    dt[, list(a)]
    或
    dt[, .(a)] # .()等同list()

  • 取多列

    dt[, .(a,b)]
    或
    dt[,a:b] #a到b列

  • 取除a以外的列

    dt[, -a]

  • 取a列中值为B的行

    dt[a==”B”,]
    a b
    1: B 1.487069
    2: B 1.107642

  • 多个筛选条件用&连接

    dt[a==”B” & b==”5”,]

  • .N计算子集行数

    dt[a==”B”, .N]

  • 取a列中值为B的行的判断

    dt[,a==’B’]
    [1] FALSE TRUE FALSE FALSE FALSE TRUE

设置a列为索引列

setkey(dt,a)

  • 取a列中值为B的行

    dt[“B”,]

  • 取a列中值为B的行，并保留第一行

    dt[“B”,mult=”first”]

  • 取a列中值为B的行，并保留最后一行

    dt[“B”,mult=”last”]

  • 取a列中值为b的行，没有数据则为NA

    dt[“b”]

对data.table对象进行增、删、改操作

  • 增加1列，列名为c

    dt[,c:=b+2]

  • 增加2列，列名为c1,c2

    dt[,`:=`(c1 = 1:6, c2 = 2:7)]
    或者
    dt[,c(‘d1’,’d2’):=list(1:6,2:7)]

  • 删除c1列

    dt[,c1:=NULL]

  • 同时删除d1,d2列

    dt[,c(‘d1’,’d2’):=NULL]

给b赋值为30

dt[,b:=30]

  • 对a列值为B的行，c2列值值大于3的行，的b列赋值为100

    dt[a==’B’ & c2>3, b:=100]
    或> dt[,b:=ifelse(a==’B’ & c2>3,50,b)]

排序

dt[order(a,-b)] #-表示逆序, !同

分组计算

  • 对整个b列数据求和

    dt[,sum(b)]

  • 按a列分组，并对b列按分组求和

    dt[,sum(b),by=a]

  • 分组计数

    dt[, .(.N), by=”a”]
    Also
    dt[, .N, by=a]

  • 分组计算并排序

    dt[, .N, by=.(a)[order(a)]]

  • .SD可按by分组取子集，同时不显示by的分组

    dt[, lapply(.SD,mean), by=a]

  • .SDcols选取特定列进行函数计算

    dt[,lapply(.SD,mean), by=.(a), .Sdcols=.(b)]

  • 连续操作

    dt[…][…][…] #类似管道符

多个data.table的连接操作

  • 设置score数据集，key为stuId

    setkey(score,”stuId”)

  • 设置student数据集，key为id

    setkey(student,”id”)

  • 合并两个数据集的数据

    student[score,nomatch=NA,mult=”all”] #两数据集key交集

JOIN type DT syntax data.table::merge() syntax
INNER X[Y, nomatch=0] merge(X, Y, all=FALSE)
LEFT OUTER Y[X] merge(X, Y, all.x=TRUE)
RIGHT OUTER X[Y] merge(X, Y, all.y=TRUE)
FULL OUTER - merge(X, Y, all=TRUE)
FULL OUTER WHERE NULL (NOT INNER) - merge(X, Y, all=TRUE), subset NA
